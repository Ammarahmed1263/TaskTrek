## Objective
The main objective of this project is to create a todo list application with core functionalities like adding, deleting, editing, and reordering tasks using drag-and-drop. Additionally, it includes a personal enhancement feature to improve user experience.

## Core Functionality
- **Add Task**: Allows users to add new tasks to their list.
- **Delete Task**: Enables users to remove tasks from their list.
- **Edit Task**: Provides functionality to modify existing tasks.
- **Reorder Tasks**: Users can reorder tasks using drag-and-drop functionality.

## User-Friendly Enhancement

### Background Notifications
The app includes background notifications that encourage users by informing them about the number of tasks left for the day. This feature aims to motivate users to complete their tasks by sending regular updates.

### Task Selection and Filters
- **Select/Deselect Task**: Users can select and deselect tasks to mark their progress.
- **Task Filters**: Filters to view all tasks, completed tasks, and tasks in progress.

### Display Today's Date
The app prominently displays today's date so if user doesn't know today's date he won't need to navigate elsewhere outside of the app.

### Encouraging Notifications
- **Notification Title**: "You're On Fire!"
- **Notification Description**: "Just {tasksLeft} tasks left to conquer today!"

## User Interface
The user interface is designed to be intuitive, accessible, and aesthetically pleasing. It incorporates the following elements:
- Simple and clean layout
- Easy navigation
- Responsive design
- Visual cues for task completion

## How to Run the Project
1. **Clone the Repository**:
    ```bash
    git clone <repository-url>
    cd <repository-directory>
    ```

2. **Install Dependencies**:
    ```bash
    npm install
    or
    yarn install
    ```

3. **Run the Application**:
    ```bash
    npm run android   # For Android
    npm run ios       # For iOS
    or
    yarn run android
    yarn run ios
    ```

<!-- 4. **Build for Release**:
    ```bash
    npm run build
    or
    yarn run build
    ``` -->
